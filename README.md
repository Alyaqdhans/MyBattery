# MyBattery
MyBattery is a simple Android app for Samsung users to check battery health information.
The app helps you easily view the log data from SysDump menu.

> [!IMPORTANT]
> MyBattery is not an official Samsung app.  
> Battery results depend on the log files available on the device.
